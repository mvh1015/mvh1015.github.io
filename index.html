Asteroid Avoidance

In GDS 111, my final project was to follow the text book, to create an Asteroid Avoidance game. With the HTML and JavaScript skills that I learned in class, not only did I create the program, but I added a shooting mechanic, and a scrolling background.

Six Shooter Program

In GDS 111, one of my first assignments using JavaScript was to create a Six Shooter program that used JavaScript to keep track of ammo and to make sound effects. I also styled the whole webpage using HTML and CSS.

Rock, Paper, Scissors

In GDS 111, I programmed my first working game, Rock, Paper, Scissors. The assignment required us to use JavaScript to create a computer player for the human player to play with. I decided to add two more actions to the game, Lizard and Spock, to make the game more fun.
